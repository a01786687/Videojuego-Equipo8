# Anura 

## Authors

- Renata Uruchurtu
- Carlos Rosete
- Emilio Torres

A 2D roguelite platformer where a small frog fights its way up the food chain in a swamp.

---

## How to Run

1. Clone or download this repository
2. Open the project in **VS Code**
3. Install the **Live Server** extension if you don't have it
4. Right click on `index.html` and open it with **Live Server**
5. The game will open in your browser automatically

NOTE: The game requires a local server to run, opening `index.html` directly in the browser will not work and some assets or scripts will not load.

---

## Where the Game Starts

The game opens on the **Title Screen**. Click **New Game** to start playing.

## Controls

| Action | Key |
|---|---|
| Move Left | a |
| Move Right | d |
| Aim Tongue Up | w |
| Jump | Spacebar |
| Crouch | s |
| Dash | j |
| Attack (Tongue) | i |
| Pause | Esc |

## What's Already Working

- Title screen with New Game, Log In and Settings buttons
- Randomly generated levels every run
- Frog movement: walking, jumping, crouching and dashing
- Tongue attack that damages enemies
- Enemies that patrol and chase the frog
- Health bar and damage system
- Camera that follows the frog for the sidescroller experience
- Platforms with collision
- Cave entrance at the end of the level that takes you to the boss
- Game over screen when health reaches 0
- Pause menu (currently it doesn't show anything but it will be developed more in future sprints)
- Brightness setting
- Login and register forms (UI only since backend connection still in development)
